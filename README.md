# Sistem-Irigasi-dan-Pemupukan-Cerdas-Smart-Irrigation-Fertigation-
Proyek ini bertujuan mengotomatisasi penyiraman dan pemberian nutrisi (pupuk) berdasarkan kebutuhan riil tanaman. 
Cara Kerja: Sensor kelembaban tanah, sensor NPK (nitrogen, fosfor, kalium), dan sensor pH dipasang di lapangan. Data dikirim ke cloud dan diolah untuk mengaktifkan pompa atau katup otomatis hanya saat dibutuhkan.
Contoh: Habibi Garden atau sistem berbasis ESP32/Arduino yang memantau level air dan nutrisi hidroponik, lalu memberikan pupuk secara otomatis.
Target Pengguna:1. Petani Modern 
Petani yang ingin meningkatkan efisiensi penggunaan air dan pupuk serta meningkatkan hasil panen dengan teknologi otomatis
2. Petani Hidroponik 
Orang atau bisnis yang menggunakan sistem hidroponik membutuhkan kontrol nutrisi yang presisi, sehingga sistem berbasis sensor seperti ESP32 atau Arduino Uno sangat membantu.
3.Hobiis Smart Farming / Urban Farming 
Orang yang menanam tanaman di rumah, rooftop, atau kebun kecil dengan konsep smart farming berbasis IoT.
